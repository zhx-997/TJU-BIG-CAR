# ROS2 树莓派小车硬件驱动节点开发工作日志

## 1. 概述

将树莓派上原有的 Python 硬件控制程序（通过 I2C 控制 PCF8591 DAC 输出电压，通过 GPIO 控制驱动使能和刹车）整合到 ROS2 Humble 中，使小车能够接收 `/cmd_vel` 话题，并转换为电机电压输出。最初采用 Python 节点，后因性能考虑改为纯 C++ 节点。

---

## 2. 环境与工作空间

- **硬件**：树莓派 4B（1GB 内存），Ubuntu 22.04 Server
- **ROS2 版本**：Humble
- **工作空间**：`~/car_ws`
- **功能包**：`car_control`

## 3. Python 版硬件驱动节点（可选）

### 3.1 硬件控制原始代码（供参考）

原始程序由四个文件组成，分别控制两个电机（通过不同 I2C 总线）、驱动使能（GPIO 17）和刹车（未实现，可扩展）。

**`enable.py`** – GPIO 17 控制驱动使能  
```python
from gpiozero import LED

_pin = LED(17, initial_value=False)

def drive_control(state):
    if state == 1:
        _pin.off()  # 0V 使能
    else:
        _pin.on()   # 3.3V 关闭
```

**`stop.py`** – 刹车控制（示例）
```python
# 假设 GPIO 18 控制刹车
from gpiozero import LED
_brake = LED(18, initial_value=False)

def brake_control(state):
    if state == 1:
        _brake.on()
    else:
        _brake.off()
```

**`speed.py`** – 总线 1 电机（左电机）
```python
import smbus

_bus = None
_address = 0x48
_vcc = 5.0

def init(address=0x48, bus_id=1, vcc=5.0):
    global _bus, _address, _vcc
    _bus = smbus.SMBus(bus_id)
    _address = address
    _vcc = vcc

def set_voltage(target_voltage):
    min_v, max_v = 1.0, 5.0
    target_voltage = max(min_v, min(target_voltage, max_v))
    digital_value = int(target_voltage / _vcc * 255)
    _bus.write_byte_data(_address, 0x40, digital_value)
```

**`speeda.py`** – 总线 0 电机（右电机），电压范围 0.5~3.0V，结构类似。

## 4. C++ 版硬件驱动节点（最终采用）

由于编译时树莓派内存不足，经过 swap 扩容后成功编译。节点采用纯 C++ 实现，直接通过 sysfs 操作 GPIO 和 Linux I2C 设备。

### 4.1 源代码 `src/hardware_driver.cpp`

```cpp
#include <rclcpp/rclcpp.hpp>
#include <geometry_msgs/msg/twist.hpp>
#include <fcntl.h>
#include <unistd.h>
#include <sys/ioctl.h>
#include <linux/i2c-dev.h>
#include <cstring>
#include <cmath>
#include <fstream>
#include <string>

class HardwareDriver : public rclcpp::Node
{
public:
    HardwareDriver() : Node("hardware_driver")
    {
        // 参数声明
        this->declare_parameter<int>("i2c_bus_left", 1);
        this->declare_parameter<int>("i2c_bus_right", 0);
        this->declare_parameter<int>("dac_address", 0x48);
        this->declare_parameter<double>("vcc", 5.0);
        this->declare_parameter<int>("drive_pin", 17);
        this->declare_parameter<int>("brake_pin", -1);
        this->declare_parameter<double>("max_linear_speed", 1.0);
        this->declare_parameter<double>("max_voltage", 5.0);
        this->declare_parameter<double>("angular_gain", 0.5);
        this->declare_parameter<bool>("simulate", false);

        // 读取参数
        int i2c_left = this->get_parameter("i2c_bus_left").as_int();
        int i2c_right = this->get_parameter("i2c_bus_right").as_int();
        dac_addr_ = this->get_parameter("dac_address").as_int();
        vcc_ = this->get_parameter("vcc").as_double();
        drive_pin_ = this->get_parameter("drive_pin").as_int();
        brake_pin_ = this->get_parameter("brake_pin").as_int();
        max_linear_speed_ = this->get_parameter("max_linear_speed").as_double();
        max_voltage_ = this->get_parameter("max_voltage").as_double();
        angular_gain_ = this->get_parameter("angular_gain").as_double();
        simulate_ = this->get_parameter("simulate").as_bool();

        // 电压范围（根据电机特性）
        left_min_ = 1.0; left_max_ = 5.0;
        right_min_ = 0.5; right_max_ = 3.0;

        // 硬件初始化
        if (!simulate_) {
            if (!init_gpio(drive_pin_, true)) {
                RCLCPP_ERROR(this->get_logger(), "Failed to init drive GPIO, switching to simulation");
                simulate_ = true;
            }
            if (brake_pin_ != -1 && !init_gpio(brake_pin_, true)) {
                RCLCPP_ERROR(this->get_logger(), "Failed to init brake GPIO, switching to simulation");
                simulate_ = true;
            }
            if (!init_i2c(i2c_left, i2c_right, dac_addr_)) {
                RCLCPP_ERROR(this->get_logger(), "Failed to init I2C, switching to simulation");
                simulate_ = true;
            } else {
                set_drive(false);
                set_brake(false);
                set_voltage(0.0, 0.0);
                hw_ok_ = true;
            }
        }

        if (simulate_) {
            RCLCPP_WARN(this->get_logger(), "Running in SIMULATION mode (no hardware control)");
        }

        // 订阅话题
        subscription_ = this->create_subscription<geometry_msgs::msg::Twist>(
            "/cmd_vel", 10, std::bind(&HardwareDriver::cmd_callback, this, std::placeholders::_1));
        RCLCPP_INFO(this->get_logger(), "Hardware driver node started.");
    }

    ~HardwareDriver()
    {
        if (!simulate_ && hw_ok_) {
            set_voltage(0.0, 0.0);
            set_drive(false);
            set_brake(true);
        }
        if (drive_fd_ != -1) close(drive_fd_);
        if (brake_fd_ != -1) close(brake_fd_);
        if (i2c_left_fd_ != -1) close(i2c_left_fd_);
        if (i2c_right_fd_ != -1) close(i2c_right_fd_);
    }

private:
    // ---------- GPIO (sysfs) ----------
    bool init_gpio(int pin, bool output) {
        // 导出引脚
        std::ofstream export_file("/sys/class/gpio/export");
        if (export_file.is_open()) {
            export_file << pin;
            export_file.close();
        }
        // 设置方向
        std::string dir_path = "/sys/class/gpio/gpio" + std::to_string(pin) + "/direction";
        std::ofstream dir_file(dir_path);
        if (!dir_file.is_open()) {
            RCLCPP_ERROR(this->get_logger(), "Cannot open GPIO direction file for pin %d", pin);
            return false;
        }
        dir_file << (output ? "out" : "in");
        dir_file.close();
        // 打开 value 文件
        std::string val_path = "/sys/class/gpio/gpio" + std::to_string(pin) + "/value";
        int fd = open(val_path.c_str(), O_RDWR);
        if (fd < 0) {
            RCLCPP_ERROR(this->get_logger(), "Cannot open GPIO value file for pin %d", pin);
            return false;
        }
        if (pin == drive_pin_) drive_fd_ = fd;
        else if (pin == brake_pin_) brake_fd_ = fd;
        return true;
    }

    void set_gpio(int fd, int value) {
        if (fd < 0) return;
        lseek(fd, 0, SEEK_SET);
        char buf[2] = {static_cast<char>('0' + value), '\n'};
        write(fd, buf, 2);
    }

    void set_drive(bool enable) {
        if (drive_fd_ < 0) return;
        set_gpio(drive_fd_, enable ? 0 : 1);  // 使能时输出 0V
    }

    void set_brake(bool active) {
        if (brake_fd_ < 0) return;
        set_gpio(brake_fd_, active ? 1 : 0);
    }

    // ---------- I2C ----------
    bool init_i2c(int bus_left, int bus_right, int addr) {
        std::string dev_left = "/dev/i2c-" + std::to_string(bus_left);
        std::string dev_right = "/dev/i2c-" + std::to_string(bus_right);
        i2c_left_fd_ = open(dev_left.c_str(), O_RDWR);
        i2c_right_fd_ = open(dev_right.c_str(), O_RDWR);
        if (i2c_left_fd_ < 0 || i2c_right_fd_ < 0) return false;

        if (ioctl(i2c_left_fd_, I2C_SLAVE, addr) < 0 ||
            ioctl(i2c_right_fd_, I2C_SLAVE, addr) < 0)
            return false;
        return true;
    }

    void write_dac(int fd, uint8_t value) {
        uint8_t cmd[2] = {0x40, value};
        write(fd, cmd, 2);
    }

    void set_voltage(double left_volt, double right_volt) {
        if (simulate_) {
            RCLCPP_DEBUG(this->get_logger(), "SIM: left=%.2fV, right=%.2fV", left_volt, right_volt);
            return;
        }
        if (!hw_ok_) return;
        left_volt = std::max(0.0, std::min(left_volt, left_max_));
        right_volt = std::max(0.0, std::min(right_volt, right_max_));
        uint8_t left_dig = static_cast<uint8_t>(left_volt / vcc_ * 255);
        uint8_t right_dig = static_cast<uint8_t>(right_volt / vcc_ * 255);
        write_dac(i2c_left_fd_, left_dig);
        write_dac(i2c_right_fd_, right_dig);
    }

    // ---------- 回调 ----------
    void cmd_callback(const geometry_msgs::msg::Twist::SharedPtr msg) {
        double linear = msg->linear.x;
        double angular = msg->angular.z;
        bool is_moving = (std::abs(linear) > 0.01) || (std::abs(angular) > 0.01);

        double base_voltage = (linear / max_linear_speed_) * max_voltage_;
        base_voltage = std::max(0.0, std::min(base_voltage, max_voltage_));
        double diff = angular_gain_ * angular;
        double left_volt = base_voltage + diff;
        double right_volt = base_voltage - diff;

        set_voltage(left_volt, right_volt);

        if (is_moving) {
            set_drive(true);
            set_brake(false);
        } else {
            set_drive(false);
            set_brake(true);
            set_voltage(0.0, 0.0);
        }

        RCLCPP_DEBUG(this->get_logger(),
            "cmd_vel: lin=%.2f, ang=%.2f -> left=%.2f, right=%.2f",
            linear, angular, left_volt, right_volt);
    }

    // 成员变量
    int drive_fd_ = -1, brake_fd_ = -1;
    int i2c_left_fd_ = -1, i2c_right_fd_ = -1;
    int dac_addr_;
    double vcc_;
    int drive_pin_, brake_pin_;
    double max_linear_speed_, max_voltage_, angular_gain_;
    double left_min_, left_max_, right_min_, right_max_;
    bool simulate_ = false;
    bool hw_ok_ = false;

    rclcpp::Subscription<geometry_msgs::msg::Twist>::SharedPtr subscription_;
};

int main(int argc, char **argv)
{
    rclcpp::init(argc, argv);
    auto node = std::make_shared<HardwareDriver>();
    rclcpp::spin(node);
    rclcpp::shutdown();
    return 0;
}
```

## 5. 编译与构建

### 5.1 增加 Swap 空间（解决内存不足）

树莓派 4B 1GB 内存编译时内存不足，需增加 swap。

```bash
# 创建 2GB swap 文件
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

# 永久生效
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```

### 5.2 编译命令

```bash
cd ~/car_ws
colcon build --packages-select car_control --executor sequential --cmake-args -DCMAKE_BUILD_TYPE=Release --symlink-install
```

- `--executor sequential` 强制单线程，降低内存占用。
- `-DCMAKE_BUILD_TYPE=Release` 减少调试信息。
- `--symlink-install` 符号链接安装。

## 6. 权限配置

### 6.1 启用 I2C 接口

编辑 `/boot/firmware/config.txt`（或 `/boot/config.txt`），添加：
```
dtparam=i2c_arm=on
```
重启后生效。检查设备：
```bash
ls /dev/i2c-*   # 应显示 /dev/i2c-0 /dev/i2c-1
```

### 6.2 I2C 权限

```bash
sudo usermod -a -G i2c $USER
```

创建 udev 规则 `/etc/udev/rules.d/99-i2c.rules`：
```
KERNEL=="i2c-[0-9]*", GROUP="i2c", MODE="0660"
```
重新加载：
```bash
sudo udevadm control --reload-rules
sudo udevadm trigger
```

注销并重新登录。

### 6.3 GPIO 权限

```bash
sudo groupadd gpio
sudo usermod -a -G gpio $USER
```

创建 udev 规则 `/etc/udev/rules.d/99-gpio.rules`：
```
SUBSYSTEM=="gpio", ACTION=="add", PROGRAM="/bin/sh -c 'chown root:gpio /sys/class/gpio/export /sys/class/gpio/unexport ; chmod 220 /sys/class/gpio/export /sys/class/gpio/unexport'"
SUBSYSTEM=="gpio", ACTION=="add", PROGRAM="/bin/sh -c 'chown root:gpio /sys/class/gpio/gpio* /sys/class/gpio/gpio*/direction /sys/class/gpio/gpio*/value ; chmod 660 /sys/class/gpio/gpio* /sys/class/gpio/gpio*/direction /sys/class/gpio/gpio*/value'"
```
重新加载 udev，注销重登录。

## 9. 常见问题与解决

| 问题                          | 原因                 | 解决                                       |
| ----------------------------- | -------------------- | ------------------------------------------ |
| 编译时系统重启                | 内存不足             | 增加 swap，使用 `--executor sequential`    |
| GPIO 操作失败                 | 权限不足             | 加入 gpio 组，设置 udev 规则               |
| I2C 设备打不开                | I2C 未启用或权限     | 启用 I2C 并加入 i2c 组                     |
| 节点进入模拟模式              | 硬件未连接或权限     | 连接硬件并配置权限，设置 `simulate:=false` |
| 编译警告 `warn_unused_result` | `write` 返回值未使用 | 可忽略，或添加 `(void)write(...)`          |
