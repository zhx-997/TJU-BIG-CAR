解决办法：Windows系统中管理员身份打开cmd输入：

```cpp
/*每行输入完成后均需要回车等待响应*/
diskpart//进入DISKPART>目录之下
list disk//看到磁盘信息，记住数据最贴合sd卡的磁盘编号
select disk x//编号,显示选择成功
clean//擦除sd卡分区信息，操作不可逆；直到显示清除成功
create partition primary//给磁盘创建主分区
format fs=fat32 quick//持续大概十几秒完成格式化
assign//分配可用盘符
exit//退出cmd
```

#### 3.创建第一个 ROS2 节点并验证通信（3/24 0：00-0：30）

创建工作空间

```
mkdir -p ~/car_ws/src
cd ~/car_ws
colcon build
```

创建功能包

```
cd ~/car_ws/src
ros2 pkg create car_control --build-type ament_python --dependencies rclpy geometry_msgs std_msgs
```

编写速度监听节点

- 在 `~/car_ws/src/car_control/car_control/` 下创建 `velocity_listener.py`

  ```python
  #!/usr/bin/env python3
  import rclpy
  from rclpy.node import Node
  from geometry_msgs.msg import Twist
  
  class VelocityListener(Node):
      def __init__(self):
          super().__init__('velocity_listener')
          self.subscription = self.create_subscription(
              Twist,
              '/cmd_vel',
              self.listener_callback,
              10
          )
          self.get_logger().info('Velocity listener node started, waiting for /cmd_vel messages...')
  
      def listener_callback(self, msg):
          self.get_logger().info(f'Received velocity: linear.x={msg.linear.x:.2f}, angular.z={msg.angular.z:.2f}')
  
  def main(args=None):
      rclpy.init(args=args)
      node = VelocityListener()
      rclpy.spin(node)
      node.destroy_node()
      rclpy.shutdown()
  
  if __name__ == '__main__':
      main()
  ```

  

- 添加可执行权限：`chmod +x velocity_listener.py`

- 修改 `setup.py`，在 `entry_points` 中添加：

  ```
  'velocity_listener = car_control.velocity_listener:main',
  ```

  编译与运行

```
cd ~/car_ws
colcon build
source install/setup.bash
ros2 run car_control velocity_listener
```

发布速度指令验证

在新终端中：

```

```
